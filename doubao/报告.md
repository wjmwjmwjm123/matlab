# Sigma-X病毒传播动力学建模与仿真报告
## 一、参数提取与数学定义
### 1.1 核心参数整理
所有参数均来自题目给定条件，转化为微分方程所需的速率参数如下：
| 参数符号 | 物理意义 | 数值 | 计算逻辑 |
|----------|----------|------|----------|
| $N$ | 城市总人口 | $10^7$ | 题目给定 |
| $I_0$ | 初始感染者数 | $100$ | 题目给定 |
| $\sigma_1$ | 潜伏前期→后期转移速率 | $0.25$ 天$^{-1}$ | 潜伏期前4天无传染性，速率为停留时间倒数$1/4$ |
| $\sigma_2$ | 潜伏后期→感染期转移速率 | $0.5$ 天$^{-1}$ | 潜伏期后2天有传染性，速率为停留时间倒数$1/2$ |
| $\gamma$ | 感染期→康复期转移速率 | $0.125$ 天$^{-1}$ | 正式感染期平均8天，速率为$1/8$ |
| $\beta_I$ | 单个感染者日均有效接触数 | $0.45$ 天$^{-1}$ | 日均接触15人×感染概率3% |
| $\beta_E$ | 单个潜伏后期感染者日均有效接触数 | $0.225$ 天$^{-1}$ | 传染力为正式感染期的50% |
| $\omega$ | 免疫衰减速率 | $6.67\times10^{-4}$ 天$^{-1}$ | 150天后10%概率失活，线性衰减速率$0.1/150$ |
| $v_{rate}$ | 日均疫苗接种量 | $10^5$ 人/天 | 题目给定，第30天起实施 |
| $\eta$ | 疫苗保护率 | $0.85$ | 85%接种者14天后获得免疫 |
| $\tau_{vax}$ | 疫苗接种启动时间 | $30$ 天 | 题目给定 |
### 1.2 时延问题的中间状态处理
为避免引入复杂的时滞微分方程，采用**链式舱室法**处理两个时延特性：
#### （1）潜伏期末期传染的处理
将传统SEIR的单一潜伏态拆分为两个串联舱室：
- $E_1$：潜伏前4天，无传染性，仅向$E_2$转移
- $E_2$：潜伏后2天，具有半传染性，可贡献感染力，向$I$转移
既满足总潜伏期6天的要求，又实现了潜伏后期的传染特性。
#### （2）疫苗14天生效延迟的处理
引入14个串联的接种等待舱室$U_1 \sim U_{14}$：
- 接种者从易感态$S$进入$U_1$，每天依次向下一个舱室转移，14天后从$U_{14}$转出
- 转出人群中85%进入免疫态$V$，15%未获得保护返回$S$
- 所有$U_i$人群仍为易感，感染后直接进入$E_1$
通过链式舱室模拟固定14天延迟，完全兼容常微分方程求解框架。
## 二、数学模型构建
### 2.1 状态变量定义
| 变量 | 物理意义 |
|------|----------|
| $S(t)$ | 未接种的易感人群数 |
| $E_1(t)$ | 潜伏前期（无传染性）人群数 |
| $E_2(t)$ | 潜伏后期（有传染性）人群数 |
| $I(t)$ | 正式感染期人群数 |
| $R(t)$ | 康复免疫人群数 |
| $V(t)$ | 疫苗诱导免疫人群数 |
| $U_i(t) (i=1\sim14)$ | 接种后第$i$天的未免疫人群（仍易感） |
### 2.2 辅助变量与干预逻辑
1. **管控系数$c(t)$**：实现迟滞式动态干预，避免阈值振荡：
   - 初始未管控：$c=1.0$，接触人数为初始水平
   - 严格管控：当$I(t)/N > 1\%$时触发，$c=0.25$，接触人数降75%
   - 松动管控：当处于管控期且$I(t)/N < 0.1\%$时触发，$c=0.5$，接触人数恢复至初始50%
   - 迟滞规则：管控触发后不再回到初始无管控状态，仅在严格/松动间切换
2. **感染力$\lambda(t)$**：考虑双传染源贡献的单位易感者感染率：
   $$
   \lambda(t) = \frac{c(t) \left( \beta_E E_2(t) + \beta_I I(t) \right)}{N}
   $$
3. **日接种人数$v(t)$**：第30天起接种，不超过剩余易感人群规模：
   $$
   v(t) = \begin{cases}
   0, & t < 30 \\
   \min(10^5, S(t)), & t \geq 30
   \end{cases}
   $$
### 2.3 微分方程组
$$
\begin{cases}
\displaystyle \frac{dS}{dt} = \omega R + \omega V + (1-\eta) U_{14} - \lambda S - v \\[6pt]
\displaystyle \frac{dE_1}{dt} = \lambda \left( S + \sum_{i=1}^{14} U_i \right) - \sigma_1 E_1 \\[6pt]
\displaystyle \frac{dE_2}{dt} = \sigma_1 E_1 - \sigma_2 E_2 \\[6pt]
\displaystyle \frac{dI}{dt} = \sigma_2 E_2 - \gamma I \\[6pt]
\displaystyle \frac{dR}{dt} = \gamma I - \omega R \\[6pt]
\displaystyle \frac{dV}{dt} = \eta U_{14} - \omega V \\[6pt]
\displaystyle \frac{dU_1}{dt} = v - U_1 - \lambda U_1 \\[6pt]
\displaystyle \frac{dU_i}{dt} = U_{i-1} - U_i - \lambda U_i, \quad i=2\sim13 \\[6pt]
\displaystyle \frac{dU_{14}}{dt} = U_{13} - U_{14} - \lambda U_{14}
\end{cases}
$$
### 2.4 人口守恒验证
所有状态变量的导数之和为0，满足总人口守恒：
$$
\frac{d}{dt}\left( S + E_1 + E_2 + I + R + V + \sum_{i=1}^{14} U_i \right) = 0
$$
## 三、仿真实现说明
采用MATLAB `ode45`变步长求解器，仿真时间跨度200天，输出步长0.1天。核心实现特性：
1. 采用`persistent`持久化变量存储管控状态，实现干预逻辑的迟滞效应，避免阈值振荡；
2. 内置有干预/无干预两组对照仿真，自动统计峰值信息；
3. 输出标准化可视化曲线，包含全人群演变和干预效果对比。
